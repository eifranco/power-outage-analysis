# When the Lights Go Out: An Analysis of Major Power Outages

**By:** Adam Hamadene, Emiliano Franco

## Introduction

This project analyzes major power outages in the continental United States from January 2000 to July 2016. The goal of this project is to understand what characteristics are associated with more severe outages, especially in terms of outage duration.
